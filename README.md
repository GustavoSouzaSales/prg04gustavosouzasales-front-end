# 🔥 UltraGás Web

Sistema web desenvolvido para gerenciamento e solicitação de gás de cozinha de forma rápida, moderna e intuitiva.

O projeto possui uma interface moderna inspirada na identidade visual da Ultragaz, oferecendo login, área administrativa e sistema de pedidos online.

---

# 🎯 Objetivo

* Facilitar pedidos de gás online
* Modernizar o atendimento ao cliente
* Centralizar gerenciamento de usuários e pedidos
* Melhorar a experiência do consumidor
* Simular uma plataforma profissional de distribuição de gás

---

# 🚀 Funcionalidades

* 🔐 Sistema de Login
* 👤 Cadastro e autenticação de usuários
* 📦 Área de pedidos
* 🛠️ Painel administrativo
* 📱 Interface responsiva
* 🎨 Design moderno inspirado na Ultragaz
* ⚡ Navegação dinâmica com React Router
* 📋 Validação de formulários
